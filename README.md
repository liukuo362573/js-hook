## js-hook
JavaScript hook any method and property helper

## 静态和非静态
js 对象方法，包括静态的和非静态的，静态的方法用 类 直接调用，非静态的方法用 类['prototype'] 进行调用，对象属性是类似的

## 局部变量
js-hook 不能 hook 局部变量，可以使用 monitor_local_variable.js 对变量的值进行监视
